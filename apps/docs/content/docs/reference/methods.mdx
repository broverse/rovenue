## Identity

| Method | Signature (RN) | Returns | Throws |
|--------|---------------|---------|--------|
| `currentUser` | `currentUser()` | `User` | â€” |
| `identify` | `identify(appUserId)` | `void` | â€” |
| `logOut` | `logOut()` | `void` | â€” |

`currentUser()` is a **cache read** â€” instant, no network. It returns a [`User`](/docs/reference/types#user) with two fields:
- **`rovenueId`** â€” permanent per-install identifier (prefix `rov_`); always present.
- **`appUserId`** â€” your backend's user ID; `null` until `identify()` is called.

`identify(appUserId)` performs a **server-side bind**: Rovenue attaches the `appUserId` label to the current `rovenueId` and, if that `appUserId` already exists on another install, automatically transfers that install's purchases and credits to this device. The call is **optimistic** â€” local state updates immediately and the server sync completes in the background. See the [Identifying users guide](/docs/guides/identifying-users) for the full flow.

`logOut()` mints a fresh `rovenueId`, clears `appUserId`, and drops all scope-bound caches. Call it when your user signs out so the next user on a shared device starts with a clean slate.

<Tabs groupId="platform" persist items={['React Native', 'Swift', 'Kotlin']}>
  <Tab value="React Native">
    ```ts
    const user = await Rovenue.currentUser();
    // user.rovenueId â€” always present (e.g. "rov_01abc...")
    // user.appUserId â€” null until identify() has been called

    // After your auth flow completes:
    await Rovenue.identify('user_123');

    // When your user signs out:
    await Rovenue.logOut();
    ```
  </Tab>
  <Tab value="Swift">
    ```swift
    let user = await Rovenue.shared.currentUser()
    // user.rovenueId â€” always present (e.g. "rov_01abc...")
    // user.appUserId â€” nil until identify() has been called

    // After your auth flow completes:
    try await Rovenue.shared.identify("user_123")

    // When your user signs out:
    try await Rovenue.shared.logOut()
    ```
  </Tab>
  <Tab value="Kotlin">
    ```kotlin
    val user = Rovenue.shared.currentUser()
    // user.rovenueId â€” always present (e.g. "rov_01abc...")
    // user.appUserId â€” null until identify() has been called

    // After your auth flow completes:
    Rovenue.shared.identify("user_123") // suspend

    // When your user signs out:
    Rovenue.shared.logOut() // suspend
    ```
  </Tab>
</Tabs>

<Callout type="warn">
Use **opaque, unguessable** IDs for `appUserId` â€” never an email address or username. Because `identify()` can transfer purchases between installs, a guessable ID is an account-takeover surface.
</Callout>

## Purchases

| Method | Signature (RN) | Returns | Throws |
|--------|---------------|---------|--------|
| `getOfferings` | `getOfferings()` | `{ current: Offering \| null; all: Record<string, Offering> }` | `NetworkUnavailableError`, `ServerError` |
| `purchase` | `purchase(pkg)` | `PurchaseResult` | `PurchaseCancelledError`, `PurchasePendingError`, `ProductNotAvailableError`, `StoreProblemError`, `ReceiptInvalidError` |
| `restorePurchases` | `restorePurchases()` | `void` | `NetworkUnavailableError`, `ServerError` |

<Tabs groupId="platform" persist items={['React Native', 'Swift', 'Kotlin']}>
  <Tab value="React Native">
    ```ts
    import {
      Rovenue,
      PurchaseCancelledError,
      PurchasePendingError,
    } from '@rovenue/react-native-sdk';

    const offerings = await Rovenue.getOfferings();
    const pkg = offerings.current?.packages[0];

    if (pkg) {
      try {
        const result = await Rovenue.purchase(pkg);
        // result.entitlements / result.creditBalance are up to date
      } catch (e) {
        if (e instanceof PurchaseCancelledError) { /* user dismissed */ }
        else if (e instanceof PurchasePendingError) { /* Ask-to-Buy */ }
        else throw e;
      }
    }

    await Rovenue.restorePurchases();
    ```
  </Tab>
  <Tab value="Swift">
    ```swift
    let offerings = try await Rovenue.shared.getOfferings()
    if let pkg = offerings.current?.packages.first {
      do {
        let result = try await Rovenue.shared.purchase(pkg)
        // result.entitlements / result.creditBalance are up to date
      } catch is PurchaseCancelledError { /* user dismissed */ }
        catch is PurchasePendingError   { /* Ask-to-Buy */ }
    }

    try await Rovenue.shared.restorePurchases()
    ```
  </Tab>
  <Tab value="Kotlin">
    ```kotlin
    val offerings = Rovenue.shared.getOfferings()
    val pkg = offerings.current?.packages?.firstOrNull()

    if (pkg != null) {
      try {
        val result = Rovenue.shared.purchase(pkg)
        // result.entitlements / result.creditBalance are up to date
      } catch (e: PurchaseCancelledException) { /* user dismissed */ }
        catch (e: PurchasePendingException)   { /* deferred */ }
    }

    Rovenue.shared.restorePurchases()
    ```
  </Tab>
</Tabs>

## Account Token

| Method | Signature (RN) | Returns | Throws |
|--------|---------------|---------|--------|
| `getAppAccountToken` | `getAppAccountToken()` | `string` (UUID) | `Storage` |

Returns a stable UUID scoped to the current user. Pass it to StoreKit's `.appAccountToken(uuid)` or Play Billing's `setObfuscatedAccountId()` before purchase. Calling `identify()` changes the scope, so the next call returns a new UUID for the known user.

<Tabs groupId="platform" persist items={['React Native', 'Swift', 'Kotlin']}>
  <Tab value="React Native">
    ```ts
    const token = await Rovenue.getAppAccountToken();
    ```
  </Tab>
  <Tab value="Swift">
    ```swift
    let token = try await Rovenue.shared.getAppAccountToken()
    ```
  </Tab>
  <Tab value="Kotlin">
    ```kotlin
    val token = Rovenue.shared.getAppAccountToken()
    ```
  </Tab>
</Tabs>

## Change Stream

Listen for real-time state changes from the SDK. The event type is one of `EntitlementsChanged`, `IdentityChanged`, or `CreditBalanceChanged`.

<Tabs groupId="platform" persist items={['React Native', 'Swift', 'Kotlin']}>
  <Tab value="React Native">
    ```ts
    const unsub = Rovenue.addChangeListener((event) => {
      switch (event) {
        case 'EntitlementsChanged':
          // Re-read entitlements
          break;
        case 'IdentityChanged':
          // User identity updated
          break;
        case 'CreditBalanceChanged':
          // Re-read credit balance
          break;
      }
    });

    // To unsubscribe:
    unsub();
    ```
  </Tab>
  <Tab value="Swift">
    ```swift
    // changes is an AsyncStream<ChangeEvent>
    for await event in Rovenue.shared.changes {
        switch event {
        case .entitlementsChanged:
            // Re-read entitlements
        case .identityChanged:
            // User identity updated
        case .creditBalanceChanged:
            // Re-read credit balance
        }
    }
    ```
  </Tab>
  <Tab value="Kotlin">
    ```kotlin
    // changes is a SharedFlow<ChangeEvent>
    Rovenue.shared.changes
        .onEach { event ->
            when (event) {
                ChangeEvent.ENTITLEMENTS_CHANGED -> { /* re-read entitlements */ }
                ChangeEvent.IDENTITY_CHANGED     -> { /* user identity updated */ }
                ChangeEvent.CREDIT_BALANCE_CHANGED -> { /* re-read balance */ }
            }
        }
        .launchIn(lifecycleScope)
    ```
  </Tab>
</Tabs>
